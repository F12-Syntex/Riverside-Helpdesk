Purpose

This protocol was intended for use in any of the 7 practices within the Hackney Downs PCN as a way of keeping up-to-date with patients’ blood tests on high risk medications, to adhere to CQC requirements and best practice. The monitoring indices below in Table 1 are intended to follow a combination of NICE guidance, the BNF, SPS and SPC monitoring recommendations.

Protocol

At the beginning of each calendar month, pharmacy technician to run pre-made searches that are in-built within EMIS as follows:

1. Go to population reporting -> EMIS library -> SNOWMED searches - > EMIS Clinical Utilities -> Drug Monitoring.

2. Click “run” to initiate all of the searches in this folder. This should be done on the first day of the month in practice.

3. The number of patients in each search category (see Table 1) should be cross-referenced with the High Risk Drug Monitoring Spreadsheet which is kept on the shared drive. Any new patients should be added to the spreadsheet at this time.

4. Work through the patient lists and record whether bloods are up-to-date (Y) or not (N). Check Cerner Portal to see if bloods have been done. If present – add note to say bloods on Cerner and date done. Nb. Specifically for DOAC monitoring, the creatinine clearance must be calculated at this time if it is due according to the monitoring indices and providing there is an up-to-date weight. If there is no such weight, patients must be recalled.

5. For those that are due their bloods, open a consultation on EMIS, change consultation type to “administration”, code as “high risk drug monitoring” in the problems and change this to an administrative problem.

6. If bloods are overdue, order the relevant bloods via T-quest and save the consultation. This is done so that requested bloods are visible to all that access the patient’s record and acts as a safety net/prompt in case of missed notifications.

7. In the first instance, send an AccuRx message to the patient informing them of the need to book in bloods via reception, using the pre-made text template. If patient does not have a mobile number on record, then you must ring the patient on their landline and attempt to book them in or leave a message asking them to book in for blood tests.

8. The following week (i.e. week 2 of the calendar month), check the spreadsheet and update accordingly. For patients that did not respond to the first round of text messages, and attempt must be made to contact them directly – 2nd recall.

9. In the third week of the calendar month, if patients have not responded to any of the above, then a letter inviting the patients in for routine blood tests should be sent to their home address – see high risk drug monitoring template invite letter.

Caveats

  1. STABLE patients only
  2. Based on MAXIMUM monitoring interval

Monitoring frequency will be different for:

    • Unstable patients
    • Elderly
    • Newly started
    • Dose change
    • Poor renal function
    • Extreme weights
    • This is only a GUIDE – patients should be reviewed on an individual basis
